## 项目简介

**福彩AI助手**是一款纯工具型福彩信息查询小程序，用户通过 AI 对话问答快速获取开奖信息、历史数据分析和玩法规则，**无需注册登录即可使用**。

产品定位为"数据查询助手"，而非"选号工具"或"预测平台"。通过自然语言对话，彩民可以像发微信一样提问，一句话直达结果，大幅降低中老年用户的信息查询门槛。

### 支持彩种

| 彩种 | 编码 | 说明 |
|------|------|------|
| 🎱 双色球 | `ssq` | 6红球+1蓝球，周二/四/日开奖 |
| 🔢 福彩3D | `d3` | 3位数字，每日开奖 |
| 🌈 七乐彩 | `qlc` | 7个基本号码+1特别号码 |
| ⭐ 快乐8 | `kl8` | 选一至选十，20个开奖号码 |

## 功能模块

### 🧠 AI问答中心（核心入口）

打开小程序即进入类 ChatGPT 的对话界面，支持 10 类标准问题：

- **开奖号码查询** — "最近一期双色球开奖号码"
- **号码遗漏统计** — "福彩3D号码123遗漏多少期"
- **历史走势统计** — "双色球红球01最近100期出现次数"
- **开奖公告/赛程** — "今天有哪些福彩开奖"
- **玩法规则** — "快乐8怎么玩"
- **数据对比查询** — "双色球最近10期蓝球"
- **期号相关** — "双色球2023150期什么时候开的"
- **中奖条件/奖金** — "双色球中4个红球多少钱"
- **历史期号范围** — "双色球最近30期开奖号码"
- **基础信息** — "福彩有哪些玩法"

AI 回答基于 **RAG（检索增强生成）** 架构，通过意图识别 → 数据检索 → 大模型生成 → 三层合规过滤，确保回答准确、安全、可解释。

### 🏆 开奖查询

- 最新开奖结果（4 彩种首页展示）
- 历史开奖记录（按日期/期号筛选，支持翻页）
- 开奖日历/未来开奖安排
- 开奖提醒订阅（微信服务通知，免费 2 个彩种，Pro 全 4 个）

### 📊 数据工具

| 工具 | 免费层 | Pro层 | 说明 |
|------|--------|-------|------|
| 走势图 | 30/50期 | 30/50/100/200/500期 | 折线图/柱状图，支持切换 |
| 遗漏统计 | 基础数据 | 全部数据 | 各号码遗漏期数、最大遗漏、平均遗漏 |
| 号码分布 | 基础数据 | 全部数据 | 红球/蓝球分布、和值分布、奇偶比、大小比 |
| 冷热分析 | 30期 | 自定义N期 | 最近N期热号/冷号排行 |
| 数据导出 | ❌ | CSV/Excel | 便于用户自行分析 |

### 📖 玩法规则

- 四个彩种完整玩法说明（投注规则、开奖规则、奖金设置）
- 各等奖级中奖条件与奖金对照表
- 兑奖指南（兑奖地点、期限、所需证件、注意事项）
- 常见术语解释（胆码、拖码、和值、跨度等）

### 📌 号码收藏

用户手动输入关注的号码组合并添加备注（如"守号组合1"），仅**本地存储**（不上传服务器），不生成注单、不打印、不关联任何投注行为。

- 免费层：最多 10 条
- Pro层：无限制

### ⚙️ 个人中心（无登录）

- 字体大小调节（小/中/大三档）
- 夜间模式开关
- 关于/版本信息/使用帮助
- 隐私政策说明
- 用户反馈入口
- 清除缓存

> **不含任何登录/注册/手机号绑定/实名认证/地理位置收集功能**

## 快速开始

### 环境要求

- Node.js ≥ 20.x
- MySQL 8.0
- Redis 7.0
- npm 或 pnpm

### 1. 克隆项目

```bash
git clone https://github.com/5wjphhc244-rgb/lottery-ai-assistant.git
cd lottery-ai-assistant
```

### 2. 后端启动

```bash
cd backend
npm install

# 配置环境变量（复制并修改）
cp .env.example .env
# 编辑 .env 文件，填写数据库、Redis、Kimi API 密钥等配置

# 启动开发服务器（带热重载）
npm run dev
```

后端服务默认运行在 `http://localhost:3000`

### 3. 前端启动

```bash
cd frontend
npm install

# 微信小程序开发模式
npm run dev:mp-weixin

# H5 开发模式
npm run dev:h5
```

前端 H5 开发服务器默认运行在 `http://localhost:5173` 或 `http://localhost:8080`

### 4. 数据初始化

```bash
cd backend
# 执行数据库迁移和种子数据
npm run db:migrate
npm run db:seed
```

## 数据模型

### 核心数据表

| 表名 | 说明 | 关键字段 |
|------|------|----------|
| `lottery_draws` | 开奖记录 | type, issue, draw_date, red_balls, blue_ball, numbers |

### 数据来源

| 来源 | 数据类型 | 抓取方式 |
|------|----------|----------|
| 中国福彩官网 (www.cwl.gov.cn) | 官方开奖公告、历史数据 | 定时 HTTP 请求，解析 HTML/JSON |
| 中彩网 (www.zhcw.com) | 历史开奖、统计数据 | API 接口或页面解析 |
| 新浪彩票 | 开奖数据、新闻资讯 | 备用数据源，补充校验 |

> 抓取频率 ≥ 2秒/请求，遵守 robots 协议

## 合规说明

### 绝对禁止（9项红线）

1. ❌ 售彩/代购/代兑
2. ❌ 预测/推荐号码
3. ❌ 诱导购彩
4. ❌ 收集个人信息（手机号、身份证、地理位置等）
5. ❌ 生成注单/投注方案
6. ❌ 投注计划提醒
7. ❌ 组织合买
8. ❌ 与站点利益绑定（分成/导流协议）
9. ❌ 线上选号→线下出票闭环

### AI 合规过滤（三层机制）

```
用户输入
  ↓
【第一层：意图识别拦截】→ 违规请求直接返回标准拒绝话术
  ↓
【第二层：生成结果审查】→ 关键词匹配 + 语义分析，拦截预测/诱导内容
  ↓
【第三层：人工审核兜底】→ 5% 抽样人工审核 + 用户举报 + 24小时响应
```

### 标准免责声明

> 以上数据来源于中国福利彩票发行管理中心官方开奖公告，仅供参考。彩票开奖号码为随机产生，历史数据不代表未来趋势。请理性购彩，量力而行。未成年人禁止购彩。
